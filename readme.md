## Installs

### Nvm

	curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.38.0/install.sh | bash
	source ~/.bashrc 
	nvm install 14 
	nvm use 14

### Yarn

	npm install --global yarn

### Vuex

	yarn add vuex@next --save

### Router

	yarn add vue-router@4

### Config

	npm i config


modify example.json to default.json
